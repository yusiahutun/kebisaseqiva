How to Fix the AMD Driver Timeout (TDR) Error on Windows

25 Mar 2024, 00:00

If you use an AMD graphics processor, you may encounter a driver timeout detection error. The error is triggered due to a Windows OS feature, Timeout Detection and Recovery (TDR). It checks and resets the graphics driver when the GPU fails to respond in a specified time. The error often causes the game to crash or stutter on your Windows computer. AMD has acknowledged the error on its official blog, though there is no one-click solution to fix this error yet.
